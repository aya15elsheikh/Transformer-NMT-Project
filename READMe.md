## Project Title

**Neural Machine Translation using Transformer (Seq2Seq Model)**

---

## Overview

This project implements a **Neural Machine Translation (NMT)** system using the **Transformer Encoder-Decoder architecture**.

The model is designed to translate between **Arabic and English**, leveraging modern deep learning techniques instead of traditional RNN-based approaches.

---

## Key components:

* Encoder-Decoder structure
* Multi-Head Attention
* Positional Encoding
* Feed Forward Networks

---

## Dataset

The model is trained using the following datasets:

  ara_.txt dataset

  salehalmansour english-to-arabic dataset

---

## Project Pipeline

The project is divided into several stages:

1. **Data Preprocessing**
2. **Tokenization**
3. **Sequence Processing**
4. **Transformer Model Implementation**
5. **Training**
6. **Evaluation**
7. **Inference & Demo**
